# Beauty Studio Frontend

Frontend part of the **Beauty Studio** project.  
The application is built with **React**, **TypeScript** and **Vite**.  
It includes authentication, admin dashboard, and client/procedure management.

## Tech Stack

- ⚛️ React + TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🗂️ Redux Toolkit
- 🔑 JWT Authentication
- 📡 Axios (HTTP client)
